# P300

The P300 (or P3) is an endogenous [[erp]] component: a positive [[eeg]]
deflection peaking ~250–500 ms after a rare, task-relevant stimulus,
maximal over parietal midline (Pz). Because the response reflects
cognitive categorization rather than stimulus physics, it can be
triggered visually, auditorily, or tactilely — making it one of the most
practical signals for BCI.

## How to elicit it

Use the [[oddball-paradigm]]: embed rare "target" stimuli the user is
instructed to attend to inside a stream of frequent standards. Amplitude
grows with target rarity; latency tracks how long categorization takes.
The response is largely automatic — minimal user training is required —
but requires sustained attention and degrades with fatigue.

## Using it in a BCI

The dominant application is the P300 [[bci-speller]] introduced by
Farwell and Donchin (1988). A 6×6 character matrix flashes rows and
columns in rapid succession; the user silently counts flashes of the
target character. The row and column that elicit the largest P300
identify the intended character.

Typical performance:

- **Healthy users:** ~95% accuracy at **3.4–4.3 characters/min** in the
  original Farwell-Donchin work.
- **Paralysed / locked-in users:** comparable accuracy to controls —
  making the P300 speller one of the few BCIs to deliver real-world
  communication for ALS patients.

**Why BCI designers like it:**

- Minimal training; response is automatic and robust across users.
- Works with low-density consumer-grade EEG.
- More reliable subject-to-subject than motor-imagery BCIs.

**Practical limitations to design around:**

- Slow: each character needs multiple flash sequences averaged to push
  the P300 above noise.
- Requires sustained attention — taxing over long sessions.
- Amplitude habituates; vary stimulus parameters to keep users engaged.

## P3a vs. P3b — which one you're using

Only the **P3b** is useful for spellers and classifier-style BCIs:

- **P3b** — ~300–500 ms, **parietal (Pz)** maximum, elicited by rare
  task-relevant targets. This is "the" P300 in BCI contexts.
- **P3a** — ~250–280 ms, **frontocentral** maximum, elicited by novel or
  distracting stimuli regardless of task relevance. Reflects involuntary
  attention capture — usually a confound, not a control signal.
