# BCI Speller

A BCI speller is an assistive communication system that allows a user to
select characters or words from a virtual keyboard using brain signals
alone — no muscle movement required. Spellers are among the earliest and
most clinically impactful BCI applications.

## Overview

The classical Farwell–Donchin speller (1988) presents a 6×6 character matrix
whose rows and columns flash in random order. Each flash that contains the
target character elicits a [[p300]] in the user's [[eeg]]; classifying which
row and column produced the largest response identifies the chosen character.

More recent designs use [[motor-imagery]], steady-state visual evoked
potentials (SSVEP), or hybrid approaches to improve speed and accuracy.
