# Stars&Bears

Non-profit building AI-augmented human brain interfaces to help people with neural diseases and other health issues, with the long-term vision to unlock human potential.

## Deployment

This site is hosted on GitHub Pages at [starsandbears.github.io](https://starsandbears.github.io). Any push to `main` automatically deploys.

## Project Structure

```
index.html   — Landing page
style.css    — Styles (dark theme, responsive)
app.js       — Form handling (Formspree), scroll animations
```
